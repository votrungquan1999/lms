# Lesson File Downloads

## Overview

Students can download lesson materials (PDFs, documents, code files, etc.) associated with their enrolled courses. Teachers/admins upload and manage these files.

## User Roles

- **Teacher/Admin**: Uploads and manages lesson files
- **Student**: Downloads lesson files

## Acceptance Criteria

### Teacher — Upload Lesson Files

- [ ] Teacher can upload files to a specific lesson
- [ ] Supported file types include PDF, DOCX, TXT, and common formats
- [ ] Teacher can replace or delete uploaded files
- [ ] Teacher can add a description/label to each file

### Student — Download Files

- [ ] Student can see a list of downloadable files for each lesson
- [ ] Student can download individual files
- [ ] Files are only accessible to enrolled students
- [ ] Download starts immediately upon clicking the download button
